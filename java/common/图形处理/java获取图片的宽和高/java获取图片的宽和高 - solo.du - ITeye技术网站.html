java获取图片的宽和高

    博客分类：
  • JAVA
JavaF# 

有两种要考虑的情况：

Java代码  
  1. 一种是：(图片在本地可以预览到)，方法如下：  
  2. BufferedImage bufferedImage = ImageIO.read(new File(imagePath));   
  3. int width = bufferedImage.getWidth();   
  4. int height = bufferedImage.getHeight();   
  5. 这里的imagePath形如：F:\\aa\\a.jpg  
一种是：(图片在本地可以预览到)，方法如下：
BufferedImage bufferedImage = ImageIO.read(new File(imagePath)); 
int width = bufferedImage.getWidth(); 
int height = bufferedImage.getHeight(); 
这里的imagePath形如：F:\\aa\\a.jpg

 

 

Java代码

  2. Image img = Image.getInstance(new URL(“http://www.baidu.com/img/baidu_logo.gif”));  
  3. System.out.println("img.width="+img.width()+" img.hight="+img.height());
